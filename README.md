# 象棋助手使用说明

这是一个给中国象棋对局用的小工具。它会看你电脑屏幕上的棋盘，识别当前局面，然后让 Pikafish 引擎计算推荐走法。

如果开启自动走棋，它还可以帮你在棋盘上点击那一步；如果不想让它自动点，也可以只看它给出的走法。

## 先看这里

使用前请准备好：

- 电脑上已经安装 Python 3。
- 棋盘画面能正常显示在电脑屏幕上。
- 本文件夹里有 `pikafish*.exe` 和 `pikafish.nnue`。

如果你只是普通 Windows 用户，通常不用管 Pikafish 版本。程序会根据 CPU 自动选择文件夹里最合适的 `pikafish-xxx.exe` 文件；如果高版本不兼容，程序会自动继续尝试低一级版本。

当前选择顺序是：`vnni512` > `avx512` > `bmi2` > `avx2` > `modern` > 无后缀。老旧电脑可以尝试 `sse41-popcnt` 或 `ssse3` 版本。

## 第一次使用

第一次用需要做两件事：

1. 安装需要的 Python 组件。
2. 校准棋盘位置和棋子图片。

### 第 1 步：打开命令窗口

在本文件夹空白处按住 `Shift`，然后点鼠标右键，选择“在终端中打开”或“在 PowerShell 中打开”。

如果你已经打开了命令窗口，请确认它的位置是在这个项目文件夹里。

### 第 2 步：安装组件

在命令窗口里输入：

```bash
python -m pip install -r requirements.txt
```

如果上面的命令提示找不到 `python`，可以试试：

```bash
py -m pip install -r requirements.txt
```

这个步骤只需要做一次。

### 第 3 步：打开你的象棋棋盘

先打开你要使用的象棋网站、软件或对局界面。

请把棋盘调整到一个固定大小，最好不要太小。校准完成以后，尽量不要再改变网页缩放、棋盘皮肤、窗口大小或显示器位置。

### 第 4 步：校准棋盘

在命令窗口里输入：

```bash
python calibrate.py
```

如果不行，试试：

```bash
py calibrate.py
```

然后按提示操作：

1. 点击棋盘左上角的交叉点。
2. 点击棋盘右下角的交叉点。
3. 看到绿色线条和棋盘格子对齐后，按 `Enter`。

注意：要点“棋盘线的交叉点”，不是点棋子的中心。

校准成功后，会生成或更新这些文件：

- `config.json`
- `templates/ref_board.png`
- `templates/` 里面的棋子图片

以后如果换了网站、棋盘皮肤、网页缩放、窗口大小，或者把棋盘移到别的显示器上，请重新校准一次。

## 平时怎么打开

校准好以后，在命令窗口里输入：

```bash
python main.py
```

如果不行，试试：

```bash
py main.py
```

打开后会出现象棋助手主窗口，默认是摆谱状态，右侧会自动打开导航。

主界面主要显示程序棋盘。点 `开始` 后，工具每次读取到屏幕棋盘，程序棋盘都会同步显示识别到的棋子。

窗口顶部有传统 Windows 风格的菜单栏：

- `棋局`：导入棋谱、保存棋谱、导入 FEN、摆谱、新棋盘、复制当前 FEN、退出。
- `连线`：开始、重新读取、设置、使用提示。
- `帮助`：检测引擎、查看关于信息。

## 菜单说明

- `开始 / 停止`：开始或停止识别棋盘。
- `重新读取`：重新读取设置，并重新计算当前局面。
- `导入棋谱`：选择 PGN、PGNS 或 TXT 棋谱文件，导入后显示棋谱最后局面。
- `保存棋谱`：手动把当前记录保存成标准 ICCS 棋谱，文件在项目目录下的 `records` 文件夹。
- `导入 FEN`：输入一段 FEN，直接载入到程序棋盘。
- `摆谱`：打开空棋盘，右键格子可添加或清空棋子。
- `新棋盘`：载入标准初始棋盘，可在程序棋盘上点击棋子走子。
- `打分图`：在右侧显示每步的开局库分数摘要。
- `导航`：默认打开，右侧“记录”区域会用中文象棋记谱显示历史步；联机时也会把我方和对方的合法走法显示到这里。导航按钮可跳到第一步、上一步、下一步、最后一步。也可以勾选红方或黑方电脑走棋，并选择时间优先或深度优先。开局库默认读取项目目录下的 `book.xqb`。
- `PV分析`：计算时显示实时 PV 分析。
- `设置`：打开设置窗口，里面有校准、自动走棋和下一局识别。
- `检测引擎`：检查 Pikafish 是否能正常启动。
- `使用提示`：打开一个简单说明窗口。
- `校准棋盘`：重新选择棋盘位置，并保存棋子识别图片。
- `复制 FEN`：复制最近一次识别到的局面。
- `退出`：关闭工具和 Pikafish 引擎。
- `保存并关闭`：保存设置窗口里的选项并关闭窗口。
- `识别下一局按钮`：在结算界面记录“下一局”按钮的位置和样子。

如果点 `开始` 时提示缺少 `config.json`，说明还没有校准，请先点 `校准棋盘`。

## 推荐使用方式

推荐先这样用：

1. 打开象棋对局。
2. 运行 `python main.py`。
3. 点 `开始`。
4. 先观察工具能不能正确识别棋盘和推荐走法。
5. 确认没问题后，再考虑开启自动走棋。

这样更稳，也更容易发现是不是棋盘位置、缩放或皮肤不合适。

## 自动走棋

自动走棋可以在 `连线 -> 设置` 里控制。

- 勾选 `自动走棋`：程序会自动点击推荐走法。
- 不勾选 `自动走棋`：程序只计算推荐走法，不会帮你点鼠标。
- `引擎核心数量`：Pikafish 使用多少个 CPU 核心，默认是 `1`。日常推荐 `1` 或 `4`；高强度对弈可以提高，但建议给系统和其他软件预留至少 `2` 个线程，例如 16 核电脑最多先试 `14`。

如果你想手动测试哪个 Pikafish 最快，可以双击某个 `pikafish-xxx.exe`。窗口能正常停留、不闪退，通常说明硬件支持。也可以在窗口里输入 `bench` 测试速度，选择性能最好且稳定的版本。

如果发现点击的位置偏上或偏下，可以调整：

```json
"auto_move_click_offset_y": -10
```

数字越小，点击位置越往上；数字越大，点击位置越往下。

## 联机等待对方走棋

点 `开始` 后，如果当前是初始棋盘并且你执黑，程序会等待红方先走；识别到棋盘变化并稳定后，才开始为我方计算和走棋。如果你执黑，并且点 `开始` 时红方已经从初始局面合法走了一步，程序会先把红方第一步记入棋谱，然后马上为黑方计算。棋局中途也可以开始连线：只要确认当前已经轮到我方走棋，点 `开始` 后程序会直接基于当前棋盘计算。

联机模式会自动记录我方和对方的合法走法，并显示在右侧“记录”区域。需要保存时，点 `棋局 -> 保存棋谱`，会把标准 ICCS 棋谱保存到项目目录下的 `records` 文件夹。

## 自动开始下一局

如果你想让工具在一局结束后自动点“下一局”，可以在 `连线 -> 设置` 里设置：

- `识别下一局按钮`：在结算界面点击“下一局”按钮中心，然后按 `Enter`。
- `自动开始下一局`：识别到下一局按钮后自动点击。

不需要这个功能的话，可以保持关闭。

## 常见问题

### 识别不准怎么办？

重新运行：

```bash
python calibrate.py
```

校准时一定要点击棋盘线交叉点，不要点棋子。

### 换了棋盘皮肤还能用吗？

通常需要重新校准。因为不同皮肤的棋子图片不一样，工具需要重新学习当前画面。

### 改了网页缩放还能用吗？

不建议改。网页缩放、窗口大小、棋盘大小变化后，最好重新校准。

### 工具点错位置怎么办？

先关闭自动走棋，把：

```json
"auto_move": false
```

确认识别正常后，再微调点击偏移，例如：

```json
"auto_move_click_offset_y": -10
```

### 使用第二个显示器怎么办？

打开 `config.json`，把：

```json
"screen_monitor": 2
```

然后重新校准。

如果想截取整个桌面，可以设为：

```json
"screen_monitor": 0
```

### Windows 上打不开怎么办？

可以按顺序检查：

1. Python 是否安装成功。
2. 是否在本项目文件夹里打开命令窗口。
3. 是否已经运行过安装命令。
4. 是否已经完成校准。
5. 文件夹里是否有 `pikafish*.exe` 和 `pikafish.nnue`。

## 打包 Windows 可运行软件

项目里已经有 GitHub Actions 打包流程：`.github/workflows/build-windows.yml`。

使用方法：

1. 把代码推送到 GitHub。
2. 打开仓库页面的 `Actions`。
3. 选择 `Build Windows app`。
4. 点击 `Run workflow`。
5. 等它运行完成后，在页面底部下载 `XiangqiHelper-windows-x64`。
6. 解压后双击 `XiangqiHelper.exe` 运行。

如果要让其他用户方便下载，发布一个版本标签即可：

```bash
git tag v1.0.0
git push origin v1.0.0
```

workflow 会自动把 `XiangqiHelper-windows-x64.zip` 上传到 GitHub `Releases` 页面，其他用户可以直接从 Releases 下载。

打包会自动带上棋盘图片、识别模板、所有 `pikafish*.exe` 和 `pikafish.nnue`。启动时程序会根据 CPU 自动选择文件夹里最合适的 Pikafish。如果你需要开局库，把 `book.xqb` 放进解压后的程序文件夹即可。

### macOS 用户注意

macOS 需要给终端程序开启“屏幕录制”权限。

如果要自动走棋，还需要开启“辅助功能”权限。

### Linux 用户注意

自动走棋建议使用 X11 桌面环境。

如果鼠标控制不可用，可以关闭自动走棋，只让工具显示推荐走法：

```json
"auto_move": false
```

## 主要文件说明

- `main.py`：打开主界面。
- `calibrate.py`：校准棋盘和棋子。
- `vision.py`：识别屏幕上的棋盘。
- `engine.py`：调用 Pikafish 计算走法。
- `xiangqi_core.py`：处理象棋规则和局面。
- `config.json`：保存设置。
- `templates/`：保存校准时截取的棋盘和棋子图片。

## 给完全不懂电脑的提醒

不要随便移动或删除本文件夹里的文件。

如果突然不能用了，最常见的原因是棋盘画面变了。先重新校准，一般就能解决。
