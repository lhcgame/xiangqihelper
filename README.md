# 象棋助手

这是一个 Windows 象棋辅助工具。它可以识别屏幕上的象棋棋盘，用 Pikafish 引擎计算走法，也可以选择自动点击走棋。

普通用户不需要安装 Python，不需要自己配置引擎，直接下载 Release 版即可。

## 下载安装

1. 打开本项目的 GitHub 页面。
2. 点击右侧或顶部的 `Releases`。
3. 下载最新版本里的 `XiangqiHelper-windows-x64.zip`。
4. 解压整个 zip 文件夹。
5. 双击 `XiangqiHelper.exe` 启动。

注意：不要只把 `XiangqiHelper.exe` 单独拖出来运行。请保留解压后的整个文件夹。

## 第一次使用

1. 打开你的象棋网站、软件或对局界面。
2. 启动 `XiangqiHelper.exe`。
3. 打开 `连线 -> 设置`。
4. 点击 `校准棋盘`。
5. 按提示点击棋盘左上角和右下角的线交叉点。
6. 校准完成后，点 `连线 -> 开始`。

校准时要点棋盘线的交叉点，不是棋子中心。

## 平时怎么用

- `连线 -> 开始`：开始识别屏幕棋盘。
- `连线 -> 停止`：停止识别。
- `连线 -> 设置`：设置自动走棋、思考时间、引擎核心数量。
- `棋局 -> 保存棋谱`：把当前记录保存到 `records` 文件夹。
- `棋局 -> 导入 FEN`：手动导入一个局面。
- `棋局 -> 摆谱`：打开空棋盘，右键格子添加棋子。
- `棋局 -> 新棋盘`：打开标准初始棋盘。

右侧“记录”区域会显示联机走棋历史和开局库分支。

## 自动走棋

自动走棋在 `连线 -> 设置` 里开启。

建议第一次使用时先不要勾选 `自动走棋`，先确认棋盘识别正确。确认没问题后，再开启自动走棋。

如果鼠标点击位置不准，请先停止使用自动走棋，然后重新校准棋盘。

## 开局库 book.xqb

如果压缩包里已经有 `book.xqb`，不用额外操作。

如果没有开局库，请把 `book.xqb` 放到 `XiangqiHelper.exe` 旁边，也就是解压后的程序文件夹根目录。

## 引擎版本

软件已自带 Pikafish 和 `pikafish.nnue`，不需要另外安装引擎。

程序会根据 CPU 自动选择合适的 Pikafish 版本。大致优先级是：

`vnni512 > avx512 > bmi2 > avx2 > modern > 无后缀`

如果高版本不兼容，程序会自动尝试低一级版本。

## 引擎核心数量

在 `连线 -> 设置` 里可以设置 `引擎核心数量`。

普通使用推荐：

- 低配电脑：`1`
- 普通电脑：`4`
- 高配电脑：可以提高，但建议至少给系统和其他软件预留 `2` 个线程

例如 16 核电脑可以先试 `14`，如果电脑卡顿，就调低。

## 常见问题

### 打不开软件

请确认你是从 Release 下载的 zip，并且已经完整解压。不要只运行单独的 exe。

### 识别不准

重新打开 `连线 -> 设置 -> 校准棋盘`。

换棋盘皮肤、网页缩放、窗口大小、显示器位置以后，都建议重新校准。

### 没有开局库

确认 `book.xqb` 是否在 `XiangqiHelper.exe` 旁边。

### 自动走棋点错

先关闭自动走棋，重新校准棋盘。确认识别正常后，再开启自动走棋。

### 保存的棋谱在哪里

点 `棋局 -> 保存棋谱` 后，棋谱会保存在程序文件夹里的 `records` 文件夹。

## 开发者说明

如果你想从源码运行，需要安装 Python 依赖：

```bash
python -m pip install -r requirements.txt
python main.py
```

Windows 打包由 GitHub Actions 完成。推送 `v*` 标签后，会自动生成 Release：

```bash
git tag v1.0.4
git push origin v1.0.4
```
